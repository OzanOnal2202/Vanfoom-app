# Capacitor assets placeholders

Vervang deze placeholders met je echte branding-assets en genereer daarna platform-assets.

## Vereiste inputbestanden

- `resources/icon.png` (1024x1024, zonder afgeronde hoeken)
- `resources/splash.png` (2732x2732, gecentreerde content)

## Aanbevolen generatie

Gebruik Capacitor assets tooling (of Android Studio/Xcode) om launcher icons en splash screens te genereren nadat je echte PNG's hebt toegevoegd.

```bash
npx @capacitor/assets generate
```
